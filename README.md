# Calcify

Calcify is a clean and responsive web-based calculator built with **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. It is designed as a minimal frontend project that demonstrates UI structuring, interactive button handling, and basic expression evaluation in the browser.

---

## Preview

Calcify provides a simple calculator interface with:

- Responsive centered layout
- Clean Tailwind CSS styling
- Number and operator inputs
- Instant expression evaluation
- Clear button to reset the display
- Basic error handling for invalid expressions

---

## Tech Stack

- **HTML5**
- **Tailwind CSS**
- **Vanilla JavaScript**

---

## Features

- Modern calculator UI
- Grid-based button layout
- Right-aligned display screen
- Operator, equal, and clear action buttons
- Expression evaluation using JavaScript
- Decimal output formatting
- Error state handling for invalid calculations
